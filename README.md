## Birthday Countdown Timer (React)

Beautiful, pink-themed birthday countdown timer for your fianc√©, targeting **22 January 2026**, built with React and Vite.

### Run locally

1. **Install dependencies**

```bash
cd /Users/macbookpro/Desktop/Private
npm install
```

2. **Start dev server**

```bash
npm run dev
```

3. Open the URL that Vite prints (usually `http://localhost:5173`) in your browser.

### Customize

- **Target date**: Update `TARGET_DATE` in `src/App.jsx`.
- **Colors & text**: Tweak the pink theme and messages in `src/styles.css` and `src/App.jsx`.
